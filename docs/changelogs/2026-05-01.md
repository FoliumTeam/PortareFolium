## v0.12.126

### fix: rendered table MDX cleanse

- `src/lib/markdown-cleanse.ts`: restores rendered `ColoredTable` HTML to canonical `<ColoredTable ... />` JSX and normalizes React attribute names to `className` / `colSpan` / `rowSpan`.
- `src/lib/markdown.tsx`: runs the same cleanse before frontend MDX render so currency `$` in table cells is not misread as KaTeX math.
- `src/components/admin/RichMarkdownEditor.tsx`: applies the cleanse on initial editor load, source-mode transitions, and update saves so DB content is repaired automatically.
- `src/__tests__/markdown-cleanse.test.ts`: adds regression coverage for rendered colored tables with stars/Korean/currency text, React attribute normalization, and fenced-code protection.
